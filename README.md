# libpbo
Library for manage PBO file of Bohemia Interactive.

## Prerequisites
yum :
```
yum install gcc-c++ make libstdc++ libssl-dev
```

apt-get :
```
apt-get install build-essential make libstdc++6 libssl-dev
```

## Build
### Build library
```
make
```
### Build testing tools
```
make test
```

## Installing
```
make install
```
