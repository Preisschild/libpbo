# libpbo
Library for manage PBO file of Bohemia Interactive.

## Prerequisites
yum :
```
yum install gcc-c++ make libstdc++
```

apt-get :
```
apt-get install build-essencials make libstdc++-6
```

## Build
### Build library
```
make
```
### Build testing tools
```
make test
```

## Installing
```
make install
```
